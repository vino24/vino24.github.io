与窗口关闭相关的几个事件

  1. window.onclose

    当在window对象上触发close事件时的事件处理器

  2. window.onunload

    当页面关闭后,会触发unload事件.

    语法： window.onunload = funcRef;

  3. window.onbeforeunload

    当窗口即将被卸载时,会触发该事件.此时页面文档依然可见,且该事件的默认动作可以被取消.
    当该事件处理函数返回一个非空的值时,会有一个弹窗提示让用户决定是否要关闭该页面.在大多数浏览器中,返回值会转换为字符串显示在弹窗上.

  1. visibilitychange

    浏览器标签页被隐藏或显示的时候会触发visibilitychange事件.

经过尝试上面的几个方案都行不通：

  1. window.close关闭时不会调用
  2. 其他几个在关闭不能使用 alert、confirm 等方法来弹窗阻止页面关闭，所有类似方法都会被 block 掉
  3. visibilitychange在微信中无法使用
  4. onbeforeunload可以弹一个默认窗口，但窗口相关信息不可配置。
